# Doorway SDK

A monorepo containing functional KMS SDK packages built on Turnkey.

## Packages

### [@doorway/core](./packages/core)

## Setup

1. Clone the repository
```bash
git clone git@github.com:OffchainLabs/doorway-kms-sdk.git
```
2. Install dependencies:
```bash
pnpm install
```
3. Build
```bash
pnpm build
```
4. Link
```bash
pnpm link
```
5. Usage locally
Run below command in another project to use it locally:
```bash
pnpm link @doorway/core
```
