# Button

A styled button with action variants.

## Import

```tsx
import { Button } from '@zerodev/wallet-react-kit'
```

## Usage

```tsx
<Button text="Connect Wallet" onClick={handleConnect} />
<Button text="Cancel" action="secondary" onClick={handleCancel} />
<Button text="Settings" action="secondaryNeutral" />
```
