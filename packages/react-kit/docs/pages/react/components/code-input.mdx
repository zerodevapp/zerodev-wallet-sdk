# CodeInput

A multi-digit OTP/code input with auto-advance, paste support, and keyboard navigation.

## Import

```tsx
import { CodeInput } from '@zerodev/wallet-react-kit'
```

## Usage

```tsx
<CodeInput
  length={6}
  onComplete={(code) => verifyOTP(code)}
/>
```
