# IconButton

A square icon-only button wrapped in a frosted glass container.

## Import

```tsx
import { IconButton } from '@zerodev/wallet-react-kit'
```

## Usage

```tsx
<IconButton iconName="check" onClick={handleConfirm} />
<IconButton iconName="check" aria-label="Confirm" disabled />
```

## Props

Extends `ButtonHTMLAttributes<HTMLButtonElement>` — all standard button attributes (`onClick`, `disabled`, `aria-label`, etc.) are supported.

| Property | Type | Description |
| --- | --- | --- |
| `iconName` | `IconName` | The icon to display. |
