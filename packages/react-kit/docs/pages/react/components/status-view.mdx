# StatusView

A centered status display with an image, title, and description text. Use it for loading, success, error, and confirmation states.

## Import

```tsx
import { StatusView } from '@zerodev/wallet-react-kit'
```

## Usage

```tsx
<StatusView imageName="loading" title="Verifying...">
  Please wait while we verify your email.
</StatusView>

<StatusView imageName="success" title="You're in!">
  Your wallet has been created.
</StatusView>

<StatusView imageName="error" title="Something went wrong">
  Please try again or choose another sign-in method.
</StatusView>
```

## Props

| Property | Type | Description |
| --- | --- | --- |
| `imageName` | `'error' \| 'loading' \| 'send' \| 'success'` | The status image to display. |
| `title` | `string` | Heading text. |
| `children` | `ReactNode` | Description text below the title. |
| `className` | `string` | Optional custom CSS classes. |
