# Text

A styled text element with consistent typography. Renders as `<p>` by default but can be changed to `<span>` or `<label>`.

## Import

```tsx
import { Text } from '@zerodev/wallet-react-kit'
```

## Usage

```tsx
<Text>Default paragraph text</Text>
<Text as="span" className="text-body1">Inline text</Text>
<Text as="label">Form label</Text>
```

## Props

Extends `HTMLAttributes<HTMLElement>` — all standard HTML attributes (`className`, `style`, etc.) are supported.

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `as` | `'p' \| 'span' \| 'label'` | `'p'` | The HTML element to render. |
